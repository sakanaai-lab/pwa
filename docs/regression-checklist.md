# 手動回帰テスト チェックリスト

> 各リファクタリング PR をマージする前に、このチェックリストを実機（PWA）で一通り確認する。
> 自動テストが薄い領域（DOM・通信・PWA）を人手で担保するための受け入れ基準。

## 実施方法

- `python3 -m http.server 8000` 等でリポジトリ直下を配信し、ブラウザで開く。
- 可能なら **変更前（main）と変更後（feature ブランチ）の両方**を別タブで開き、挙動・見た目を比較する。
- ライト（Aquarium）/ ダークの両テーマで主要画面を確認する。

## チェック項目

### チャット基本フロー
- [ ] 新規チャットを作成できる
- [ ] メッセージを送信し、ストリーミング表示で応答が返る
- [ ] 応答の停止（中断）ボタンが機能する
- [ ] メッセージの編集・再生成・カスケード（分岐）が機能する
- [ ] コードブロックのシンタックスハイライト（Prism）が表示される
- [ ] Markdown（marked）が正しくレンダリングされる

### マルチプロバイダー
- [ ] Gemini で送受信できる
- [ ] OpenAI 互換（OpenAI / Anthropic / Groq / DeepSeek / xAI / Mistral / Z.ai / OpenRouter）で送受信できる
- [ ] Amazon Bedrock で送受信できる
- [ ] ヘッダーのモデル切替セレクタが本体設定と同期する
- [ ] カスタムモデルIDの登録が機能する

### 高度な機能
- [ ] Function Calling（「絵を描いて」で NovelAI 画像生成）が呼び出される
- [ ] ファイル/画像の添付・送信ができる（サイズ上限・MIME 判定含む）
- [ ] Anthropic 非対応画像形式が正しく変換/処理される
- [ ] 思考プロセス表示（Gemini / Claude / DeepSeek-R1）が折りたたみ表示される
- [ ] チャットタイトルが自動生成される
- [ ] 会話統計パネル（ⓘ）が表示され、トークン/コスト/キャッシュヒット率が出る
- [ ] 校正機能が動作する

### データ管理
- [ ] チャット履歴の保存・読込・削除・複製
- [ ] プロファイルの作成/切替/削除（上限5）
- [ ] プロジェクトの作成/切替/削除、プロジェクト単位の履歴フィルタ
- [ ] シーン / スタイルプロファイルの切替
- [ ] 永続メモリの有効/無効と保存
- [ ] 設定の保存・復元（リロード後も保持）
- [ ] インポート / エクスポート

### Dropbox 連携
- [ ] Dropbox 認証・同期（Push/Pull）が機能する
- [ ] 競合時のマージ挙動

### PWA / 基盤
- [ ] オフライン状態でアプリが起動する
- [ ] Service Worker が更新され、新バージョン通知が出る（`CACHE_NAME` 更新時）
- [ ] 旧バージョンの IndexedDB からのアップグレードが成功し、データが保持される
- [ ] タブ間同期（BroadcastChannel）
- [ ] テーマ切替（ライト/ダーク）で表示崩れがない

## リファクタリング時の必須確認
- [ ] 配信ファイル構成を変えた場合、`sw.js` の `CACHE_NAME` と `urlsToCache` を更新した
- [ ] `index.html` / `style.css` のキャッシュバスター（`?v=...`）を更新した
- [ ] `npm run lint` のエラー数がベースライン（`docs/findings-phase0.md`）以下である
- [ ] `npm test` がグリーンである
